# Marketing Message 2

Aprende todo lo que necesitas saber para crear una aplicación con MongoDB y Python... ¡en español!

MongoDB University va a lanzar la quinta edición de M101P: MongoDB para desarrolladores Python con soporte para hispanohablantes.

-Este curso abarcará instalación básica, JSON, diseño de esquemas, consultas, inserción de datos, indexación, replicación, sharding y utilización del driver de Python.

-Construirás una plataforma de blogs sobre MongoDB. Los ejemplos de código estarán en Python.

-Todos los vídeos incluirán subtítulos en español traducidos por miembros de nuestras comunidades Masters of MongoDB y MongoDB User Group (MUG). Este esfuerzo fue organizado por Juan Carlos Farah, Jorge Puente Sarrín y José Carlos García. Nuestros agradecimientos a ellos. 

-Juan Carlos, Jorge y José Carlos servirán como asistentes de enseñanza en el foro para ayudar a hispanohablantes (y otros) si tuvieran alguna pregunta.

-Este curso es liderado por Andrew Erlichson, VP de Educación y Richard Kreuter, ingeniero consultor en MongoDB, Inc.

-Para mayor información, visita education.mongodb.com.

Las clases de M101P: MongoDB para desarrolladores Python comienzan el 9 de setiembre - Regístrate ahora
